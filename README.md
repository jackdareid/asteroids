# Asteroids 

A simple recreation of the Asteroids arcade game built in Python using Pygame.

The project was created to practice object-oriented programming, game loops, collision detection, and event-driven programming.

---

## Features

- Player controlled spaceship
- Asteroid spawning
- Asteroid splitting
- Collision detection
- Player movement and rotation
- Shotting mechanics
- Game over state

--- 

## Technologies

- Python 3
- Pygame

---

## Installation

Clone the repository:

```bash
git clone https://github.com/jackdareid/asteroids.git
cd asteroids
```

Install dependencies:

```bash
uv sync
```

Run the game:

```bash
uv run main.py
```

---

## Controls

|   w   | move forwards  | 
|   a   | rotate left    |
|   s   | move backwards |
|   d   | rotate right   |
| Space | shoot          |
